# Calculus Practice Problems

## 1. Limits

1.  Evaluate: $\lim_{x \to 3} \frac{x^2 - 9}{x - 3}$
2.  Evaluate: $\lim_{x \to 0} \frac{\sin(5x)}{x}$
3.  Evaluate: $\lim_{x \to \infty} \frac{2x^3 + 4x - 1}{5x^3 - 2x + 8}$

## 2. Derivatives

1.  Differentiate: $f(x) = x^3 \sin(x)$
2.  Find $f'(x)$ for $f(x) = \ln(x^2 + 1)$
3.  Compute the gradient of $f(x, y) = 3x^2y - y^3$ at the point $(1, -1)$.

## 3. Integration

1.  Evaluate: $\int (4x^3 - 2x + 5) dx$
2.  Evaluate: $\int_{0}^{\pi} \sin(x) dx$
3.  Evaluate: $\int x e^{x^2} dx$ using u-substitution.

---

## **Solutions**

### **Limits**
1.  **6** (Factorize $(x-3)(x+3)$)
2.  **5** (Multiply by $\frac{5}{5}$, apply $\lim \frac{\sin x}{x}=1$)
3.  **2/5** (Divide by highest power $x^3$)

### **Derivatives**
1.  **$3x^2 \sin(x) + x^3 \cos(x)$** (Product Rule)
2.  **$\frac{2x}{x^2 + 1}$** (Chain Rule)
3.  **$\nabla f = \begin{bmatrix} -3 \\ 0 \end{bmatrix}$** (Gradient Vector)

### **Integration**
1.  **$x^4 - x^2 + 5x + C$**
2.  **2** ($[-\cos(x)]_{0}^{\pi} = -(-1 - 1) = 2$)
3.  **$\frac{1}{2}e^{x^2} + C$** (Let $u = x^2$, $du = 2x dx$)
